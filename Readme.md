## ⚠️ commit 컨벤션

```
{이모지}{태그}: {커밋 메시지}
```

### 이모지 및 태그

- 이모지는 선택에 따라 활용한다.

| 이모지 | 태그       | 설명                      |
|:----|:---------|:------------------------|
| ✨   | feat    | 기본적인 커밋             |
| ♻️  | refactor | 코드 수정                |
| ✏️  | comment  | 주석 변경(코드 변경 X) |
| 📝  | docs     | README와 같은 문서 수정        |
| 🚚  | rename   | 파일, 폴더명 수정 혹은 이동        |
